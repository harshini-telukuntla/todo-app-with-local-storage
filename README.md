# ✅ To-Do App with Local Storage

A simple and interactive task management web application built using HTML, CSS, and JavaScript.

## 🚀 Features
- Add new tasks
- Mark tasks as completed
- Delete tasks
- Tasks persist after page refresh using Local Storage
- Clean and responsive UI

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript
- Browser Local Storage API

## 📌 Project Description
This project demonstrates DOM manipulation, event handling, and client-side data storage using Local Storage. It allows users to manage daily tasks efficiently while ensuring data persistence.

## 🎯 Learning Outcome
- Improved understanding of JavaScript arrays and objects
- Hands-on practice with Local Storage
- Strengthened front-end development fundamentals
